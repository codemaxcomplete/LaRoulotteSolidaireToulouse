# 📚 Dossier DOCS  
*Documentation officielle, guides, procédures et supports pédagogiques*

Ce dossier contient toute la **documentation interne et externe** du projet :  
guides bénévoles, procédures, fiches pratiques, supports imprimables, documents de formation et ressources administratives.

Il constitue la base documentaire de la Roulotte Solidaire Toulouse.

---

## 🗂️ Organisation du dossier
- Creer, éditer et monter

---

## 🧩 Types de documents attendus

### ✔️ Guides
- guide du bénévole  
- guide de communication  
- guide de logistique  

### ✔️ Procédures
- déroulé d’une maraude  
- règles de sécurité  
- gestion du matériel  
- protocole d’accueil des bénévoles  

### ✔️ Matériel
- inventaires  
- listes de besoins  
- checklists  

### ✔️ Communication
- affiches  
- flyers  
- documents publics  

### ✔️ Formation
- modules pédagogiques  
- fiches thématiques  
- supports de sensibilisation  

---

## ⚠️ Règles importantes

> ❌ Aucun nom de bénéficiaire  
> ❌ Aucune donnée personnelle  
> ❌ Aucune photo identifiable dans ce dossier  
> ❌ Pas de documents sensibles non anonymisés  

---

## 📝 Bonnes pratiques

- utiliser des fichiers `.md` pour la documentation  
- structurer avec des titres clairs  
- ajouter des sommaires si nécessaire  
- garder un ton neutre, bienveillant et accessible
