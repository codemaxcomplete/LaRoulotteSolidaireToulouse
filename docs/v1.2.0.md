# 📘 Version 1.2.0 — Documentation & Notes de mise à jour

## 🎯 Résumé
La version **1.2.0** marque une étape importante dans la structuration et l'automatisation du projet La Roulotte Solidaire Toulouse. Cette release se concentre sur la qualité du code, la documentation terrain et l'intégration continue.

## 📦 Mises à jour principales
- **Documentation** : Restructuration complète des guides, templates et politiques internes.
- **Workflows CI** : Intégration continue via GitHub Actions (`ci-dryrun.yml`) pour la validation automatique.
- **Sécurité & Confidentialité** : Renforcement des règles d'anonymisation et de conformité dans tous les processus.
- **Support Mobile** : Optimisation des scripts d'installation pour environnements Termux/Android.

## 🧪 Procédure de test
Avant de valider ou merger, assurez-vous que les vérifications suivantes sont passées :

1. **Linting Bash** : Exécutez `shellcheck scripts/mobile/*.sh`. Aucun warning critique ne doit apparaître.
2. **Simulation d'installation** : Lancez `bash scripts/mobile/install_termux_godmode.sh --profile standard --dry-run --verbose`.
3. **Vérification structurelle** : Confirmez la présence et la cohérence des dossiers `docs/`, `actions/` et `.github/`.
4. **Conformité** : Vérifiez qu'aucun nom, visage ou donnée sensible n'est présent dans les fichiers modifiés.
