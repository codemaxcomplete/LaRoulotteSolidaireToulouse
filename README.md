---

### 🎯 Vision du projet

La Roulotte Solidaire Toulouse aspire à devenir un modèle local de solidarité mobile, fondé sur :

- la proximité humaine  
- l’action directe  
- la simplicité logistique  
- la transparence  
- l’autonomie citoyenne  

Notre ambition est de créer un dispositif léger, réactif et reproductible, capable d’apporter un soutien concret aux personnes en situation de précarité, tout en mobilisant les habitants de la métropole toulousaine.

### 🧭 Objectifs du dépôt

Ce dépôt GitHub sert à :

- centraliser la documentation du projet  
- structurer les actions et procédures  
- organiser les ressources bénévoles  
- partager les supports de communication  
- assurer la transparence et la gouvernance  
- faciliter la contribution citoyenne  
- permettre la réutilisation du modèle par d’autres collectifs  

Il constitue la base documentaire officielle de la Roulotte Solidaire Toulouse.
